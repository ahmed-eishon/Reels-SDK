# Obsidian Setup Guide

This documentation uses Mermaid diagrams that render best in Obsidian.

## Quick Setup

### 1. Install Obsidian
Download from: https://obsidian.md

### 2. Open Vault
1. Launch Obsidian
2. Click "Open folder as vault"
3. Select the `docs/` folder from reels-sdk repository

### 3. View Documentation
- Files will open as markdown with rendered Mermaid diagrams
- Use Reading View (Cmd/Ctrl + E) for best rendering
- Click internal links to navigate between documents

## Alternative: VS Code

If you prefer VS Code:

### Required Extensions
- **Markdown Preview Mermaid Support**
- **Markdown All in One**

### Usage
- Open docs folder in VS Code
- Use Markdown Preview (Cmd/Ctrl + Shift + V)
- Mermaid diagrams will render in preview

## Alternative: GitHub

All documentation is also viewable on GitHub with Mermaid support enabled.

---

**Note:** Obsidian is recommended for the best documentation experience with proper diagram rendering and fast navigation.
